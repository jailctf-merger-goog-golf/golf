# golf

## running web server

run `python3 main.py` in the directory containing this README.md

## building new js stuff

create `.env` file with these contents:

```
SAFETY_KEY=(copy from discord pinned)
SECRET_KEY=(put random text here, smash your keyboard)
```

run `npm i` to install packages

modify `script.js` as you wish

run `node build.js` to build the `bundle.js` file

done
